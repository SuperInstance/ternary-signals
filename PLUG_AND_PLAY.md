# PLUG_AND_PLAY — ternary-signals

> *Integration guide.*

## Dependency

```toml
[dependencies]
ternary_signals = "0.1.0"
```

## Feature Flags

| Feature | Default | Description |
|---------|---------|-------------|
| `std` | yes | Standard library |
| `alloc` | yes | Allocator support |

## Integration

Import `ternary_signals` in your project to access the functionality.

```
use ternary_signals::{};
```

## Compatibility

- **Rust edition**: 2021+
- **Targets**: All tier-1 Rust targets
